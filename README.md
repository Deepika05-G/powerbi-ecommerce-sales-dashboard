# E-Commerce Sales & Customer Analytics Dashboard

## Overview

This project is an interactive Power BI dashboard developed using the Superstore E-Commerce dataset containing 9,994 transaction records. The dashboard provides comprehensive insights into sales performance, customer behavior, product profitability, and regional trends to support data-driven business decisions.

## Objectives

* Analyze overall sales and profit performance
* Identify top-performing customers and products
* Evaluate regional sales and profitability
* Study the impact of discounts on profit
* Build an interactive business intelligence dashboard

## Tools & Technologies

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling
* CSV Dataset
